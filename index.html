Javascript (JS)

Javascript is an object-oriented computer programming language commonly used to create interactive effects within web browsers

Operators

Operators in JavaScript are very similar to operators that appear in other programming languages. The definition of an operator is a symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

Variables

Variables can be thought of as named containers. You can place data into these containers and then refer to the data simply by naming the container. Before you use a variable in a JavaScript program, you must declare it. Variables are declared with the var.

Functions

Function Declaration can only exist as a “statement” in isolation. All it can do is create an object variable parented by its current scope.

Methods

JavaScript methods are the actions that can be performed on objects. A JavaScript method is a property containing a function definition. Property. Value.

Arguments

JavaScript functions have a built-in object called the arguments object. The argument object contains an array of the arguments used when the function was called (invoked).
